## 版本概述
本版本为基于Flutter 3.22.0适配的OpenHarmony版本。本版本支持和完善OpenHarmony平台侧能力，提升稳定性。

## 新增特性
- 无障碍支持xcomponent多实例/多引擎场景
- 修改engine编译依赖的仓库的管理方式

## Bug修复
- 修复输入法文本光标位置更新和文本错误替换的问题
- 修复外接物理键盘时,用中文输入法输入内容后按删除键,导致额外删除输入框中的字符的问题
- 修复输入框导致的闪动的问题
- 修复arkts接口用number类型保存指针时，指针高位数据被截断的问题

## 版本发布时间
2025年6月7日

## 版本配套
- OpenHarmony API17
- ROM: 5.0.1.120
- IDE: DevEco Studio 5.0.13.200
- Flutter SDK: 3.22.0-ohos-1.0.3
