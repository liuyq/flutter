## Version Overview
This version is an OpenHarmony version based on Flutter 3.22.0. This version supports and improves the capabilities of the OpenHarmony platform and improves stability.

## New Features
- Accessibility support for xComponent multi-instance/multi-engine scenarios
- Change the management approach of repositories that engine compilation depends on

## BugFix
- Fixed the issue of updating the text cursor position and replacing text errors in the input method
- Fixed the problem where, when using an external physical keyboard, pressing the delete key after entering content with the Chinese input method would result in additional characters being deleted from the input box
- Fixed the flickering issue caused by the input field
- Fixed the bug that when the pointer is stored in the number type in the arkts interface, the high-order bits of the pointer is truncated

Jun 7, 2025
